# CSCI460: Heart Failure Risk Clustering

## Overview
CSCI460 Fall 2025 Programming Project III: Applies K-means clustering to heart failure dataset to identify patient subgroups and map to disease outcomes (0: Normal, 1: Heart Disease). Includes K selection, algorithm advantages/limitations analysis.

## Features
- K-means clustering with elbow method & silhouette score
- Cluster visualization (PCA/TSNE)
- Disease outcome mapping & purity evaluation
- Preprocessing pipeline for clinical features

## Quick Start
pip install -r requirements.txt
python clustering.py
